## 交通資訊

### 1. 郵輪

| 項目 | 內容 |
|------|------|
| 出發 | 4/24（五）14:00 基隆出發 |
| 抵達 | 4/25（六）13:00 抵達那霸若狹埠頭 |
| 通關時間 | 約 30-60 分鐘 |
| 碼頭 → Kukuru House | 步行 15-20 分鐘 |

### 2. 沖繩單軌電車（Yui-Rail）

- 最近站：美榮橋站（步行 2 分鐘）
- 單線系統，簡單好用

**票價：**

| 類別 | 一日券價格 |
|------|-----------|
| 大人 | ¥800 |
| 兒童（6-12 歲） | ¥400 |
| 未滿 6 歲 | 免費 |

**主要搭乘路線：**

- 美榮橋 ↔ 首里（首里城）
- 美榮橋 ↔ 奧武山公園
- 美榮橋 ↔ 那霸機場

### 3. 租車（OTS）

| 項目 | 內容 |
|------|------|
| 營業所 | OTS 泊營業所（步行 10 分鐘） |
| 費用 | 約 ¥8,250+/天（含保險） |
| 建議租車天數 | 2 天（Day 2 北上 + Day 3 中南部） |
| 駕照要求 | 國際駕照（日內瓦公約格式）或台灣駕照日文譯本 |
| ETC 卡 | 可租借 |

### 4. 計程車

| 項目 | 內容 |
|------|------|
| 路線 | Kukuru House → 那霸機場 |
| 費用 | ¥1,400-1,800 |
| 時間 | 15-20 分鐘 |
| 計費方式 | 起跳 ¥560，每 365m 加 ¥70 |

### 5. 機場交通方式比較

| 方式 | 費用 | 時間 | 適合情境 |
|------|------|------|---------|
| 計程車 | ¥1,400-1,800 | 15-20 分 | 帶小孩 + 行李最方便 |
| 單軌 | ¥340/大人 | 約 20 分 | 美榮橋 → 那霸空港 |
| 巴士 | ¥240/大人 | 約 30 分 | 最便宜但帶行李不便 |
