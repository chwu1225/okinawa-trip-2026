| **適合年齡** | 所有年齡 |

## 付費景點

### 8. 沖繩美麗海水族館（美ら海水族館）

| 項目 | 資訊 |
|------|------|
| **地址** | 沖繩縣國頭郡本部町（海洋博公園內） |
| **門票** | 大人 ¥2,180 / 6 歲 ¥610 / 2 歲免費 |
| **16:00 後優惠** | 大人 ¥1,510 |
| **營業時間** | 8:30 - 18:30（夏季延長至 20:00） |
| **交通（開車）** | 從那霸出發約 1.5 小時，有停車場 |
| **交通（公車）** | 可達但需 2 小時以上，推薦開車 |
| **適合年齡** | 所有年齡 |
| **建議停留** | 2 - 3 小時 |

**特色亮點**
- 黑潮之海：世界最大級水槽之一
- 鯨鯊、鬼蝠魟等大型海洋生物
- 水族館旁設有免費兒童樂園（海洋博公園內）
- 沖繩必訪地標

**省錢提示**
- 16:00 後入場可享優惠票價，適合安排下午場次

## 玩水資訊（4 月下旬）

| 項目 | 說明 |
|------|------|
| **海灘開放** | 4/25 大部分海灘已開海 |
| **海水溫度** | 約 21 - 23 度 C（偏涼） |
| **2 歲幼童** | 不建議下海游泳，海邊踏浪可以 |
| **推薦玩水地點** | 兒童王國 Splash Park（水深 20cm、水溫較暖、4 月已開放、2 歲安全） |
| **備註** | 海邊踏浪沒問題，但 2 歲以下不建議下水游泳 |
