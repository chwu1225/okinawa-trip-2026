## 重點提醒

### 廉航行李策略
- **推薦虎航**：手提 10kg/人（樂桃僅 7kg），家庭總額度更寬裕
- **推車免費託運**：樂桃一律免費；虎航僅限 2 歲以下同行
- **建議加購 1 件 20kg 託運**：網路預購約 NT$950，很划算
- **盡快訂票**：4/29 接近黃金週，越晚越貴

### 早餐安排
| 日期 | 方案 | 理由 |
|------|------|------|
| Day 1 (4/25) | 船上/便利商店 | 下午才到 |
| Day 2 (4/26) | 自煮 | 早起取車效率最高 |
| Day 3 (4/27) | 自煮 | 同上 |
| Day 4 (4/28) | WBF Art Stay Buffet ¥600 | 無車日慢慢吃 |
| Day 5 (4/29) | 泊港漁市場 | 最後一餐吃好的！ |

### 信用卡
- **CUBE 卡「趣旅行」方案**：Airbnb 享 3% 回饋（扣手續費淨回饋 1.5%）

### 天氣 (4/25-4/29)
- 白天 25-27°C / 夜間 19-21°C
- 海水 21-23°C（偏涼，2 歲不建議下水）
- 推薦玩水：兒童王國 Splash Park（水深 20cm、水溫暖）
